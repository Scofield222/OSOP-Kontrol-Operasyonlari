# OSOP-Kontrol-Operasyonlari
Açık Kaynak İşletim Sistemleri dersinde bu projede servisleri kurmak için bir kontrol paneli oluşturmak için kullanılır.

# Projenin Amacı
-başlat/durdur/yeniden başlat işlemleri
-etkinleştirip ve devre dışı bırakma
-servis şablonunu (unit files) görüntüleme ve düzenleme

# Gerekli Sistem Teknolojileri
- Ubuntu
- Linux
- Python
